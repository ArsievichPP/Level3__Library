## **Level3: library  SH++**

### **Requirements:**<br>
PHP: 8.0.2<br>
MySQL - 8.0.23 <br>
Apache/2.4.46 <br>

### **Getting Started**<br>
* clone the repository to your host
* enter your data in:
  /config/configDB.php 
* enter your data in settings:
  /config/settings.php
* run migrations (path: sql/run_migrations.php)
* add sql/soft-delete.php and sql/backup.php in cron list


### **Add admin or change username / password:**
/config/admins.php
